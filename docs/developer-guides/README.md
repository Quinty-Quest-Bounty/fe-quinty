## Tech Stack

**Frontend:**
- Next.js 14 + React 18 + TypeScript
- Tailwind CSS + shadcn/ui components
- Wagmi v2 + Viem + RainbowKit

**Smart Contracts:**
- Solidity 0.8.28
- OpenZeppelin libraries
- Base Sepolia (Chain ID 84532)

**Storage:**
- IPFS for decentralized file storage
- Pinata as pinning service
